Debug Steps

  1. Check if the application is running on http://localhost:8080
  2. Verify the notifications table exists in your database
  3. Check if there are notifications with the correct user email
  4. Test the API endpoints using the buttons above
  5. Check browser console for any JavaScript errors
  6. Verify CORS is working (check Network tab in DevTools)
